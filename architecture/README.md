# Architecture Interview Preparation

These notes are written for senior and architect-level interviews around JavaScript, TypeScript, React, Node.js, Express, MongoDB, Redis, Agentic AI, CI/CD, Docker/Kubernetes, and AWS.

Each technology file contains 15 frequently asked interview questions with practical, production-focused answers.

Use this answer structure in interviews:

```txt
What it is
Where I used it
How I implemented it
Why I chose it
Trade-offs
Failure cases
How I monitored/debugged it
```
